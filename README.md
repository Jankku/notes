# Notes

Simple notes Android app in Kotlin.

![Screenshots](screenshots.png)

## Features

- Free
- Open source
- Light/dark theme
- Export/import notes
- Pin favorite notes
- No permissions
- No ads/trackers

## Tech stack

- Kotlin
- Coroutines
- Jetpack
  - Room
  - Navigation
  - ViewModel
  - LiveData
  - Preference
